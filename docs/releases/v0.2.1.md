# Release v0.2.1 — CLI version display fix

## Summary

- `ai-passport --version` now shows the **npm package version** (e.g. `0.2.1`), not the passport schema version (`1.0.0`)
- `ai-passport info` labels passport format as **Schema** instead of **Version**

## Install

```bash
npm install -g @ai-passport-core/cli@0.2.1
```

No functional changes to memory provider, MCP, or passport schema.
