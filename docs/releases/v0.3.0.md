# Release v0.3.0 — Memory deepening (RFC 0007 Phase D)

## Summary

- **Confidence decay** — effective confidence halves every 90 days without re-verification
- **`memory verify`** — reset decay clock; optional confidence update
- **`memory query`** — `--min-confidence`, results sorted by effective confidence
- **Knowledge graph** — `memory link`, `memory graph`, MCP `get_memory_graph`
- Graph queries require `knowledge` namespace in consumer grant

## Install

```bash
npm install -g @ai-passport-core/cli@0.3.0
```

## Quick start

```bash
ai-passport memory store knowledge '{"entity":"TypeScript","role":"primary language"}' --confidence 0.9
ai-passport memory store preferences "Prefers minimal diffs" --confidence 0.85
ai-passport memory link <pref-id> <knowledge-id> prefers
ai-passport grant cursor --memory preferences,knowledge --yes
ai-passport memory query cursor --min-confidence 0.5
ai-passport memory graph cursor --root <knowledge-id>
ai-passport memory verify <record-id> --confidence 0.95
```

## MCP (Cursor)

New tool: `get_memory_graph` (requires `knowledge` in grant).

Restart MCP after global install.
