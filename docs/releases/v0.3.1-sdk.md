# @ai-passport-core/sdk v0.3.1

Aligned with `@ai-passport-core/cli@0.3.1`.

## What's included

- Same `Passport.load()` / `export()` API as CLI SDK export
- Memory Provider (RFC 0007) — `export()` returns `memory` block when grant includes `--memory` namespaces
- Confidence decay, graph records — via shared core (no new SDK methods; use CLI `memory` commands for writes)

## Install

```bash
npm install @ai-passport-core/sdk@0.3.1
```

## vs CLI package

| Package | Version | Use when |
|---------|---------|----------|
| `@ai-passport-core/sdk` | 0.3.1 | Apps — lighter deps (ajv, keytar) |
| `@ai-passport-core/cli` | 0.3.1 | CLI + MCP + same SDK at `./sdk` |
