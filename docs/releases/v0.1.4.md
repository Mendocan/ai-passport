# Release v0.1.4 — VS Code extension + readiness CLI

## Summary

- **VS Code extension** at `extensions/vscode/` — Setup Workspace, Configure MCP, Show Status
- `ai-passport readiness --consumer vscode --json` for tooling
- VS Code MCP format: `vscode_mcp_config` in readiness output (`.vscode/mcp.json`)
- Updated `examples/mcp.vscode.json` to VS Code `servers` schema

## CLI

```bash
npm install -g @ai-passport-core/cli@0.1.4
ai-passport readiness --consumer vscode --json
```

## VS Code extension (local)

```bash
cd extensions/vscode
npm run compile
```

Command Palette → **AI Passport: Setup Workspace**

Guide: [VSCODE_SETUP.md](https://mendocan.github.io/ai-passport/VSCODE_SETUP.html)
