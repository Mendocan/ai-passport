# Release v0.2.0 — Standalone SDK package (Phase 5)

**Title suggestion:** `v0.2.0 — @ai-passport-core/sdk`

## Summary

- New npm workspace: `@ai-passport-core/sdk` at `packages/sdk/`
- Same `Passport.load()` API — lighter deps (no Commander, no MCP)
- CLI `@ai-passport-core/cli/sdk` import path unchanged (backward compatible)
- Build: `npm run build:all` · Test: `npm run test:sdk-package`

## Install

```bash
npm install @ai-passport-core/sdk
```

## Quick start

```typescript
import { Passport } from '@ai-passport-core/sdk';

const passport = await Passport.load();
const context = await passport.export('my-app');
```

Full API: [SDK.md](https://mendocan.github.io/ai-passport/SDK.html)

## Publish (maintainers)

```bash
npm run build:all
npm publish --workspace=@ai-passport-core/sdk
```
